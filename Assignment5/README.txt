Without Slider:
Please put the images in the TestImages Folder. The run the Assignment5.cpp file.

SliderCode:
Prerequisite: Please make sure that the image to be read is in the same directory as in the 'main.cpp' file

Running: Run the executable, and use slider for choosing the size of the Structuring element and Operation as following:

Operations:
	0.	Erosion
	1.	Dilation
	2.	Opening
	3.	Closing

SE (Structuring Element):
	0.	1x2 all 1s matrix [1,1]
	1.	3x3 Diamond matrix
	2.	3x3 All 1s matrix
	3.	9x9 All 1s matrix
	4.	15x15 All 1s matrix
